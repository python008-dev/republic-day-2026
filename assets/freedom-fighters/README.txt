FREEDOM FIGHTERS IMAGES FOLDER
================================

Upload the following images in this folder:

1. gandhi.jpg (or .png) - Mahatma Gandhi
2. bhagat-singh.jpg (or .png) - Bhagat Singh
3. bose.jpg (or .png) - Subhas Chandra Bose
4. rani-lakshmibai.jpg (or .png) - Rani Lakshmibai
5. ambedkar.jpg (or .png) - Dr. B.R. Ambedkar
6. patel.jpg (or .png) - Sardar Vallabhbhai Patel

IMAGE REQUIREMENTS:
- Format: JPG, PNG, or WebP
- Recommended size: 400x400px or larger
- Aspect ratio: Square (1:1) works best for card display

NOTE: If images are not found, the cards will automatically show 
the first letter of the freedom fighter's name as a fallback.
